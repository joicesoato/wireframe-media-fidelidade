# 💼 Portfólio UI/UX — Projeto One Page

Este projeto foi desenvolvido como atividade prática da formação em **UI/UX Designer da DIO (Digital Innovation One)**, com o objetivo de aplicar conceitos de design de interface, experiência do usuário e desenvolvimento front-end.

---

## 🚀 Sobre o projeto

O projeto consiste em um **site de portfólio one page**, desenvolvido para apresentar:

* Informações pessoais e profissionais
* Habilidades e competências
* Projetos e cases
* Formulário de contato

A proposta foi transformar um **wireframe inicial** em uma interface visual moderna, limpa e funcional, seguindo boas práticas de UI/UX.

---

## 🎯 Objetivos

* Aplicar conceitos de **UX (User Experience)**
* Trabalhar **hierarquia visual e tipografia**
* Desenvolver um layout responsivo (estrutura base)
* Criar uma interface com foco em **clareza e usabilidade**
* Praticar **HTML e CSS na construção de interfaces reais**

---

## 🧠 Conceitos aplicados

* Wireframe → Interface final
* Design System básico
* Grid layout
* Espaçamento e alinhamento
* Hierarquia visual
* Componentização (cards, botões, seções)
* UX Writing (estrutura de conteúdo)

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

---

## 🎨 Layout

O layout foi inspirado em interfaces modernas de portfólios digitais, com foco em:

* Minimalismo
* Clareza visual
* Destaque para conteúdo
* Navegação simples e intuitiva

---

## 📸 Seções do projeto

* **Hero**: apresentação principal
* **Sobre mim**: descrição profissional
* **Habilidades**: competências técnicas
* **Projetos**: cases desenvolvidos
* **Contato**: formulário e redes sociais

## 🌐 Como visualizar

Você pode abrir o projeto localmente:

```bash
# Rodar servidor local
python3 -m http.server 8000
```

Ou simplesmente abrir o arquivo `portifolio_desktop_1440.html` no navegador.
